## How It Works

Each session, the agent reads active plans from daily notes and knowledge graph, then pre-analyzes each item:
- What's blocking it?
- What's the minimum input needed from the human?
- What can be executed autonomously right now?

Result: no "so what needs doing?" — the agent already knows, already analyzed, already has a plan ready.
